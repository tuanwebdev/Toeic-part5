# MỆNH ĐỀ QUAN HỆ (RELATIVE CLAUSES) VÀ MỆNH ĐỀ QUAN HỆ RÚT GỌN

Dưới đây là hệ thống kiến thức toàn diện về **Mệnh đề quan hệ (MDQH)** và **Mệnh đề quan hệ rút gọn**, được thiết kế ngắn gọn, trực quan để dễ học và áp dụng khi làm bài.

---

# PHẦN 1: MỆNH ĐỀ QUAN HỆ (RELATIVE CLAUSES)

Mệnh đề quan hệ dùng để bổ nghĩa cho danh từ đứng trước nó.

## 1. Các đại từ quan hệ

### Who
- Thay cho danh từ chỉ **người**.
- Làm **chủ ngữ** hoặc **tân ngữ**.

**Cấu trúc:**

```text
N (người) + who + V
N (người) + who + S + V
```

### Whom
- Thay cho danh từ chỉ **người**.
- Chỉ làm **tân ngữ**.

**Cấu trúc:**

```text
N (người) + whom + S + V
```

### Which
- Thay cho **vật/sự vật**.
- Làm cả **chủ ngữ** và **tân ngữ**.

**Cấu trúc:**

```text
N (vật) + which + V
N (vật) + which + S + V
```

### That
- Thay thế được cho **who**, **whom**, **which** trong mệnh đề quan hệ xác định (không có dấu phẩy).

### Whose
- Chỉ sự sở hữu của cả người và vật.

**Cấu trúc:**

```text
N1 + whose + N2 + V
N1 + whose + N2 + S + V
```

---

## 2. Các trạng từ quan hệ

### When
- Thay cho mốc **thời gian**.
- Tương đương:

```text
on which
in which
at which
```

### Where
- Thay cho mốc **nơi chốn**.
- Tương đương:

```text
in which
at which
on which
to which
```

### Why
- Thay cho **lý do**.
- Thường đứng sau **the reason**.
- Tương đương:

```text
for which
```

---

# PHẦN 2: MỆNH ĐỀ QUAN HỆ RÚT GỌN (REDUCED RELATIVE CLAUSES)

Khi rút gọn, ta bỏ đại từ quan hệ và biến đổi động từ chính theo 3 trường hợp cốt lõi sau.

---

## 1. Rút gọn dạng chủ động → Dùng V-ing

Áp dụng khi:
- Đại từ quan hệ đóng vai trò chủ ngữ.
- Mệnh đề mang ý nghĩa chủ động.

### Công thức

```text
Who/Which/That + V
→ V-ing
```

### Ví dụ

**Câu gốc:**

```text
The man who stands over there is my teacher.
```

**Rút gọn:**

```text
The man standing over there is my teacher.
```

---

## 2. Rút gọn dạng bị động → Dùng V3/V-ed

Áp dụng khi mệnh đề mang ý nghĩa bị động.

### Công thức

```text
Who/Which/That + be + V3/V-ed
→ V3/V-ed
```

### Ví dụ

**Câu gốc:**

```text
The book which was written by Nam is a local best-seller.
```

**Rút gọn:**

```text
The book written by Nam is a local best-seller.
```

---

## 3. Rút gọn dạng cụm động từ nguyên mẫu → Dùng To V

Áp dụng khi danh từ đứng trước có:

### Số thứ tự

```text
the first
the second
the third
...
```

### So sánh nhất

```text
the best
the most
the shortest
...
```

### Từ chỉ tính duy nhất

```text
the only
the last
...
```

### Công thức

```text
Who/Which/That + V
→ To V

Who/Which/That + be + V3/V-ed
→ To be + V3/V-ed
```

### Ví dụ

**Câu gốc:**

```text
He is the last man who left the room.
```

**Rút gọn:**

```text
He is the last man to leave the room.
```

# Mẹo làm bài trắc nghiệm nhanh (TOEIC/THPT)

Khi thấy câu đã có động từ chính chia thì hoàn chỉnh nhưng vẫn còn một vị trí trống cần điền động từ:

### Trường hợp 1

Phía sau có **tân ngữ (danh từ)**

→ Chọn **V-ing** (chủ động)

### Trường hợp 2

Phía sau là **giới từ** (by, in, on, at,...) hoặc đứng cuối câu

→ Chọn **V3/V-ed** (bị động)

### Trường hợp 3

Danh từ phía trước có:

```text
the first
the last
the only
so sánh nhất
```

→ Chọn **To V**

# Một số cấu trúc cần nhớ

### Each

```text
Each + N (số ít)
```

**Ví dụ:**

```text
Each student has a book.
```

### All / Some / Most

```text
All + N số nhiều / N không đếm được
Some + N số nhiều / N không đếm được
Most + N số nhiều / N không đếm được
```

**Ví dụ:**

```text
Most students passed the exam.
```

```text
Some water is left.
```

---

# Lưu ý về "that"

Sau **that** thường là một mệnh đề hoàn chỉnh:

```text
S + V + ...
```

Ví dụ:

```text
I think that he is right.
```

Nếu muốn diễn tả hành động đóng vai trò chủ ngữ, thường dùng:

```text
V-ing
```

hoặc

```text
To V
```

Ví dụ:

```text
That he arrived late surprised everyone.
```

```text
Arriving late surprised everyone.
```

```text
To arrive late surprised everyone.
```
